# Room(Kotlin) - Room 예제

Kotlin으로 제작한 Room 예제입니다.

예제 기능은

- EditText에 이름을 입력한다.
- 추가 버튼 클릭시 Room에 해당 이름 저장 및 TextView에 DB 데이터 표시
- 삭제 버튼 클릭시 Room 내부 정보 전체 삭제
